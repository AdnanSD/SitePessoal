Bootstrap Grid System

Aprenda a usar o sistema de colunas do Bootstrap

Como funciona o Grid?

O sistema de grid do Bootstrap é baseado em 12 colunas. Você pode usar classes como col-* para definir o número de colunas em diferentes tamanhos de tela.

Exemplo 1: 3 colunas iguais

Coluna 1
Coluna 2
Coluna 3

Exemplo 2: Colunas com larguras diferentes

Coluna 1 (4/12)
Coluna 2 (8/12)

Exemplo 3: Colunas responsivas

Coluna 1
(6/12 em telas pequenas, 4/12 em médias)
Coluna 2
(6/12 em telas pequenas, 8/12 em médias)

Desenvolvido com Bootstrap 5
